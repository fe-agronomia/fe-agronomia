# Hi, I'm Fernando 👋

**Market intelligence | Agricultural commodities | Quantitative analysis in R**

I work in market intelligence for the Brazilian sugar-energy and agricultural commodities sector — sugar, ethanol and grains. My work combines crop, weather and market data analysis with reporting and pipeline automation, turning raw data into clear communications for trading and analyst audiences.

---

### 🔭 What I work on

- **Quantitative analysis** of sugarcane crush metrics (ATR, TCH, mix), ethanol-gasoline parity and pricing dynamics
- **Automated R pipelines** running on a Linux server: API ingestion, web scraping, and parameterized Markdown reports
- **Agro-meteorological monitoring**: rainfall outlooks and frost-risk alerts for production regions
- **Bilingual market communications** (PT/EN) for internal and external distribution
